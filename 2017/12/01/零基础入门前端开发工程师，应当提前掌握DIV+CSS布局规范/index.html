全局样式设置

1.全局样式置顶，并用注释标注为/全局样式/

2.用一内容定义的样式写在一起，并用注释标注

3.一个样式里面，多个属性的定义建议写在一行，属性定义之间用”；”隔开

4.样式中类或者 id 选择器的命名建议使用样式命名规范来书写

5.自定义选择器名称建议都用小写、尽量使用英文单词，少用拼音，名称中建议少用”-“或者”_“,尽量少用英文缩写。

d.标签属性需要用引号，且不能简写，如:
